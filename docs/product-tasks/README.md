## Логика

- `P0` — задачи, которые напрямую повышают вероятность успешного звонка уже на текущей модели продукта.
- `P1` — сильные улучшения качества опыта, но не самые первые блокеры для запуска семейного сценария.
- `P2` — стратегический продуктовый сдвиг, который стоит делать после стабилизации базового опыта.
