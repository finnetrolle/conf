# Задача: Добавить режим доступности для старшей аудитории

## Приоритет

`P1` — задача очень важная, но часть пользы можно получить раньше за счет базовых текстовых и мобильных улучшений из `P0`.

## Контекст

Приложение должно быть удобным для быстрых видеозвонков родителям, включая пользователей со слабым зрением, низкой цифровой уверенностью и привычкой ориентироваться на крупные и явные элементы интерфейса.

## Проблема

Иконки без подписей, мелкие элементы и перегруженные состояния снижают понятность. Даже если базовый интерфейс выглядит современно, он может быть психологически и визуально сложным для старшей аудитории.

## Что нужно сделать

- Спроектировать "упрощенный режим" интерфейса для семейных звонков.
- Сделать ключевые кнопки крупнее и снабдить их текстовыми подписями.
- Повысить контраст и читаемость текстов и состояний.
- Свести к минимуму число одновременных действий на экране.
- Проверить сценарий на соответствие базовым требованиям доступности: размер таргетов, контраст, фокус, понятные подписи.

## Почему это важно

Если пользователь не уверен, на что нажимать, он чаще завершает сценарий или зовет кого-то на помощь. Для семейного продукта уверенность и спокойствие пользователя важнее визуальной компактности.

## Критерии готовности

- Для основных действий звонка предусмотрены крупные кнопки с текстовыми подписями.
- На экране нет критически важных действий, представленных только иконкой без расшифровки.
- Контраст текста и элементов соответствует читаемому и спокойному сценарию использования.
- Упрощенный режим можно включить как основной интерфейс для старшей аудитории.
- Проведена хотя бы базовая проверка на доступность для мобильного и десктопного сценариев.
